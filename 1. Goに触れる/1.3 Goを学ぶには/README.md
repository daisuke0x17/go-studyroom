## Goを学ぶ理由
- 多くのプロダクトで採用
- チーム開発に向いている
  - 静的型付け
    - レビューコストが低い 
    - コンパイルでエラーを見つけられる
  - 文法がシンプル
- パフォーマンスが良い
  - ゴールーチンとチャネル
  - バイナリになる
