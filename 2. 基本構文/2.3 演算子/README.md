## 代入演算子

| 演算子 |    説明     |  利用例   |
|:---:|:---------:|:------:|
| :=  | 変数の初期化と代入 | a:=100 |
| ++  |  i+=1と同義  |  i++   |
| --  |  i-=1と同義  |  i--   |

※`i++`と`i--`は式ではなく文
```go
func main(){
	var i int
	i++
	
	fmt.Println(i++) // できない
	++i // できない
}
```
